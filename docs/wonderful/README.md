# Wonderful — UI Inspiration

Sem tomaj nahráva screenshoty / obrázky UI ktoré sa mu páčia ako inšpirácia pre agentx.

Nejde o copy-paste dizajn — chceme zachovať shadcn default look. Tieto obrázky slúžia ako podklad pre diskusie o layoute, informačnej hierarchii, interakčných vzoroch a UX detailoch (ako by to **mohlo** vyzerať / fungovať).

## Ako s nimi pracovať

- Claude si ich pozrie pri návrhu wireframeov a UX rozhodnutí
- Ak vznikne zo screenshotu konkrétny pattern do `15-ux-guidelines.md`, spomeň v komente ktorý obrázok ho inspiroval
- Staré / neaktuálne obrázky môžeme zmazať, nie je to archív
